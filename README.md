## Lancer le jeu (v1.3.0.0 et au dessus)
Il suffit de lancer le launcher.bat que vous allez trouver en dézippant le jeu, ce script va dététer la dernière version disponible et la télécharger au besoin.

### Comment contribuer :
[lire ceci](https://docs.google.com/document/d/1fa3D19WODOGfzuFB0nntnGh6Ka1HNWWl7MmxHykPdjk/edit?tab=t.0)

### Créer des releases
Pour créer des releases il faut la structure suivante :
version.txt (contiens la version de la release)
Launcher.bat (doit être inclus dans chaque release mais quand on l'a ca sera la base de lancement du jeu)
Game (dossier qui contiens les fichiers du build)
